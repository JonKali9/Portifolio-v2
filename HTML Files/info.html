Resources

Codecademy

Great resources for learning all kinds of programming, however is a paid for service.

Codewars

Offers you a plethora of coding challenges which can take you anywhere from a few minutes to a few hours.

Freecodecamp

Free and very easy to use, offer plenty of Python material as well as other stuff too.

Repl

For hosting and running code online and on the go. Great for collaborating.

TryHackMe

Great site for everything Cyber Sec, great for beginners and has a free version too.

Treehouse

Great for people thinking about getting into Programming.

Udemy

Thousands of great courses, although each one needs to be paid for seperately.

W3Schools

Full of tutorials on Web Dev as well other topics.

Youtube

Great free platform for all kinds of stuff, plenty of videos here.
